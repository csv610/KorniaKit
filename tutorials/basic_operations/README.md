# Basic Operations

Introduction to fundamental image processing operations using Kornia.

## Tutorials

### readimg.py
Demonstrates basic image reading and loading using Kornia. Learn how to:
- Load images from disk
- Convert to tensor format
- Display image properties

**Usage:**
```bash
python readimg.py
```

### flip.py
Explores image flipping and mirroring operations. Learn how to:
- Flip images horizontally
- Flip images vertically
- Flip along different axes

**Usage:**
```bash
python flip.py
```

### resize.py
Master image resizing and scaling with Kornia. Learn how to:
- Resize images to specific dimensions
- Maintain aspect ratio
- Use different interpolation methods

**Usage:**
```bash
python resize.py
```

## Key Concepts

- **Tensor Operations**: Kornia works with PyTorch tensors (GPU-accelerated)
- **Batch Processing**: Operations can process multiple images at once
- **GPU Support**: All operations can run on GPU for better performance
